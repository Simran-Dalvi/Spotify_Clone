# Spotify_Clone


Netflix's KPop Demon Hunters (2025) won two Oscars at the 98th Academy Awards in March 2026. The film took home the awards for Best Animated Feature Film and Best Original Song for the hit song "Golden".
It is the first K-pop song to win an Oscar, with directors Maggie Kang and Chris Appelhans.

Since this movie is such a entertaining and heartwarming. Not only having a good story but also representing a different culture. with respect to the music industry and learning purpose i chose to make a spotify clone for this but with only the songs for this movie.
 
this is a front end app.
that is the web page plays the songs of the movie kpop demon hunter.

This project is to make a spotify clone using HTML, CSS, JavaScript and run songs in it.


the page has:
interactive play pause button
list / tiles of songs
